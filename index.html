mbitcoinjs-lib

mbitcoinjs-lib is a library of extensions to the bitcoinjs-lib bitcoin/crypto javascript library; it is designed to provide extended functionality without interfering with any pre-existing code that calls into bitcoinjs-lib.

Overview

Additions to Bitcoin.Wallet, Bitcoin.Transaction, Bitcoin.Script namespaces to support multi-signature and data/memo outputs.

Creates transactions containing multisig (M of N) and data outputs ("OP_RESERVED <80 bytes>").

Recognizes multisig as spendable if the required M addresses are in the wallet, signs/spends with M keys.

Additions to Bitcoin.Address namespace for brainwallet passphrase support, also vanity address miner.

New namespace Bitcoin.ImpExp for importing/exporting data in BBE/BCI JSON text formats to/from wallets.

Some primary functions (doc in source)

  • Bitcoin.Wallet.createSend2(): new version of createSend with multisig and data output support, also supports multiple outputs and async mode.

  • Bitcoin.Wallet.selectOutputs(): determines which spendables will be redeemed for a pending spend transaction and calcs helpful stats.

  • Bitcoin.Wallet.queryOutputs(): searches an output dataset.

  • Bitcoin.Address.fromPrivOrPass(): resolves a private key or brainwallet passphrase into address/pubkey/etc data structure, also makes random passphrases from user supplied or rfc1751 dictionary.

  • Bitcoin.ImpExp.BBE.import(): imports BBE/BCI JSON text data into a wallet.

  • Bitcoin.ImpExp.BBE.export(): exports wallet to BBE text.
